// Java program to illustrate if-else statement 

class IfElseDemo 
{ 

    public static void main(String args[]) 

    { 

        int i = 10; 

  

        if (i < 15) 

            System.out.println("i is smaller than 15"); 

        else

            System.out.println("i is greater than 15"); 

    } 


OUTPUT-:

i is smaller than 15
